## Scope

The 1.13 release target is syntax-aware evidence packets:

```text
sensors/tokmd/
  manifest.json
  analyze.md
  analyze.json
  context.md
  syntax.json
```

The release claim is narrow:

```text
tokmd can package scoped analysis, context, and optional syntax receipts into a
manifest-first evidence packet with advisory review priority.
```

The release does not claim undefined behavior detection, safety proof, public
reachability proof, semantic call graph coverage, type resolution, CI proof, or
merge readiness.
